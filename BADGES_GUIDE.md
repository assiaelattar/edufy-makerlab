# Makerlab Badge System Guide

This guide outlines the recommended badge system for the Makerlab ERP, based on the configured stations and skills. Use this reference when setting up badges in the "Setup Factory" -> "Badges" section.

### How to Add Badges
1. Go to **Learning View** (Studio).
2. Click on **Setup** tab.
3. Select **Badges** sub-tab.
4. Click **New Badge** and use the details above to populate the form.
